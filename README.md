Fhy健康助手/FhyHealthAssistantLite

简介

这是一款定时提醒小工具，可以设置待办、持续工作休息、宿舍场景的断电提醒等，提醒您在使用设备时注意休息。主界面背景为必应每日一图，界面设计使用了粘土风格。大部分代码为ChatGLM生成。

Fhy听力精灵/FhyHearingAssistantLite

简介

一款防止玩竞技类游戏大喊大叫的小工具，可以设置分贝阈值和弹窗次数，如果超过分贝就会弹窗，弹窗次数超过设定就会结束设置的进程。退出软件需要输入密码（虽然只是君子密码，通过其他方式还是可以关掉的）大部分代码为Kimi生成。
